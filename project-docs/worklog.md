# WORKLOG MODULE

## 기능 개요

사내 업무일지 입력 및 조회 기능.

---

## 입력 구조

### 전일업무
어제 수행 업무 입력.

### 금일업무
오늘 예정 업무 입력.

---

## 입력 항목

- 시간
- 장소
- 업체명
- 장비명
- 주요업무
- 비고

---

## 현재 기능

- 입력
- 저장
- 수정
- 삭제
- 날짜 조회
- 팀별 조회
- 상태 표시

---

## 조회 기능

- 카드형 사용자 목록
- 작성 여부 표시
- 문서형 상세 조회
- 수정/삭제 가능

---

## 현재 UX 방향

- 한 화면 가독성 최우선
- ERP 느낌 제거
- SaaS 스타일 유지
- 표 구조 유지
- 주요업무 강조

---

## 완료 사항

- 초 단위 제거
- 팀장 우선 정렬
- 본인만 수정 가능
- 문서형 모달 개선

---

## 향후 계획

- 업체명 자동완성
- 장비명 자동완성
- 외근 데이터 연동 검토
- 통계 기능 검토
