# SCHEDULE MODULE

## 기능 개요

사내 일정 관리 기능.

---

## 현재 기능

- 월간 캘린더
- 일정 등록
- 일정 삭제
- 일정 종류 구분
- 사용자 이름 표시
- 일정 색상 구분
- 출장 기간 자동 생성

---

## 일정 종류

- 출장
- 외근
- 회의
- 연차
- 기타

---

## 현재 UX 방향

- 달력은 최대한 심플
- 이름만 노출
- 상세는 클릭 시 모달
- 색상으로 일정 종류 구분

---

## 완료 기능

- 본인 일정만 수정/삭제 가능
- overflow 처리
- 다중 사용자 표시
- 기간 출장 삭제 지원

---

## 향후 계획

- 결재 연동
- 휴가 자동 등록
- 출장 승인 연동
- 업무일지 외근 연동 검토
